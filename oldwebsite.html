My Old Website


This is my previous personal website. It served a similar purpose as a portfolio for my work and skills. As good as the website looks on the surface, the code behind it isn't the most maintainable, the projects page is difficult to navigate through as I complete more of them, and my desired aesthetic has changed. These are a few of the reasons why I decided to start from scratch with this new personal webiste, but I would still like to acknowledge and appreciate this project.

Technologies/skills employed:

HTML: forms the structure of the webpage
CSS: manages all the styling, thematic elements, fonts, etc.
Visual Studio Code: IDE used for the development of this website
Github Pages: used to host this site publicly
Creativity and Design: Since this was a personal project, I had total freedom to explore my desired flow of the webpage, fonts, theme, layout, etc.
